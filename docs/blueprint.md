# **App Name**: Umbral: Empathetic Companion

## Core Features:

- Mood Logging: Log daily mood with emoji-based selection and subtle haptic feedback.
- Personalized Recommendations by Cero: AI-powered bot 'Cero' suggests habits and readings tailored to the user's current mood using the Genkit tool.
- Habit Management: Create, customize, and track daily habits with simple switch controls and progress visualization; data stored in Firestore.
- Emotional Library: Curated content library with search and filter options for various emotional states and content types, offering PDFs or audio with an embedded player.
- Progress Visualization: Visualize mood patterns, habit completion rates, and achievements through interactive charts and graphs.
- User Authentication: Secure user authentication system for personalized data management via Firebase.

## Style Guidelines:

- Primary background: Soft Cream (#FFF8E1) for a calm and clean aesthetic.
- Primary accent: Deep Indigo (#3F51B5) for primary actions and highlights.
- Secondary color: Soft Lavender (#E6E0FF) for UI elements and backgrounds.
- Call-to-action highlight: Soft Coral (#FFAB91) for interactive buttons.
- Body and headline font: 'Literata' (serif) for a literary, calming feel.
- Use simple, line-based icons to represent habits, emotions, and content types.
- Employ a card-based layout to organize content and habits; focus on a clear hierarchy with ample spacing.
- Implement subtle animations, such as the breathing animation of Cero, to enhance user engagement without distraction.
